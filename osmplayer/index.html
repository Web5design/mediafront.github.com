Overview

The Open Standard Media Player (OSM) Player is not just another HTML5 media player. It is a media player framework that allows tight integration into multiple 3rd party media solutions such as YouTube and Vimeo. It is not only completely themeable, but is also completely customizable using an Object Oriented plugin mechanism that gives you control over every aspect of playable web media.
